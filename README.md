## DJ Roomba: a macOS-native proto-music player

I will have more to say about this in the future. 

For now, what I can say is that it plays music. How I know this is, 
at 4 different points last night while we were watching Project Hail Mary, Claude spontaneously
had this thing blasting the Red Hot Chili Peppers out of my office. 

Despite that obvious sign from God, I've persisted in bringing this into being.
